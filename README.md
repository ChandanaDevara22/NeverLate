## 📋 Project Overview

**LifeSaver AI** is a proactive productivity companion built for the **Coding Ninjas Vibe2Ship Competition**. When you're running late, it autonomously finds your meeting in Google Calendar, drafts a polite email via Gmail, and suggests a new time - all with one click.

### Key Features:
- 🚗 **"I'm Late" Agent**: Auto-detects delays, finds meetings, drafts emails
- 📝 **Task Management**: Natural language tasks with priority detection
- 📅 **Google Calendar**: Real-time event creation and search
- 📧 **Gmail Integration**: One-click draft creation
- 🎨 **Sophisticated Dark Theme**: Bento grid with glassmorphism

### The Problem:
Students, professionals, and entrepreneurs frequently miss deadlines and important commitments. Existing tools rely on passive reminders that are easy to ignore.

### The Solution:
An AI agent that proactively assists users in planning, prioritizing, and completing tasks before deadlines are missed.

## 📁 Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   ```bash
   npm run dev
   ```
